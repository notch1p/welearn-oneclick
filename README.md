## **WelearnCurriculumFinsh**

## 特别鸣谢Avenshy和SSmJaE大佬提供的v0.1-4版本
  

## 简介
* 输入账号密码或cookie登录并一键完成课程
* 可以选择课程
* 可指定每章节正确率
* 已完成和未开放的章节会自动跳过
